# \# STHIRA

# 

# Structural engineering platform for steel and timber member design, reporting, member schedules, and project management.

# 

# \---

# 

# \## Current Release

# 

# \*\*Version:\*\* v1.1.0

# 

# \### Available Features

# 

# \- Steel Beam Design

# \- Steel Column Design

# \- Timber Beam Design (EC5)

# \- Eurocode Load Combinations (G + Q + Wind + Snow)

# \- Automatic Floor Load Take-Down

# \- Automatic Bearing Wall Load Take-Down

# \- Member Schedule Generation

# \- PDF Report Generation

# \- Excel Export

# \- Project Save / Load

# \- Section Recommendation Engine

# \- Reinforcement Calculation for Slof Supports
