# STHIRA

Structural engineering platform for steel member design, reporting, member schedules, and project management.

## Features

- Steel Beam Design
- Steel Column Design
- Eurocode Load Combinations
- Member Schedule Generation
- PDF Report Generation
- Excel Export
- Project Save / Load
- Section Recommendation Engine
- Reinforcement Calculation for Slof Supports

## Technology Stack

- Python
- FastAPI
- HTML
- CSS
- JavaScript

## Current Modules

### Steel Beam

- Bending verification
- Shear verification
- Deflection verification
- Section recommendation
- Slof design support

### Steel Column

- Utilization checks
- Governing axis evaluation
- Section recommendation

## Reports

- Steel Beam PDF Report
- Steel Column PDF Report
- Member Schedule PDF
- Excel Workbook Export
