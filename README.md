# STHIRA

Structural engineering platform for steel and timber member design, reporting, member schedules and project management.

## Current Release

**v1.1.0**

### Features

- Steel Beam Design
- Steel Column Design
- Timber Beam Design (EC5)
- Eurocode Load Combinations (G + Q + Wind + Snow)
- Automatic Floor & Wall Load Take-Down
- PDF Reports
- Excel Export
- Project Save / Load
- Member Schedule Management

### Current Modules

#### Steel Beam
- Bending
- Shear
- Deflection
- Wind & Snow combinations
- Section recommendation

#### Steel Column
- Compression checks
- Utilization checks
- Section recommendation

#### Timber Beam (EC5)
- Bending
- Shear
- Deflection
- Serviceability
- Automatic load take-down
- PDF reporting

### Design Standards

- EN 1990
- EN 1991
- EN 1993 (Eurocode 3)
- EN 1995 (Eurocode 5)

### Reports

- Steel Beam PDF
- Steel Column PDF
- Timber Beam PDF
- Member Schedule PDF
- Excel Workbook Export
